# Product Guidelines: openQuote

## 1. Visual Identity & UI Design
*   **Established Design:** The project adheres to an existing UI design provided by the developer. New features must integrate seamlessly with the current aesthetic.
*   **Utility-First:** The interface prioritizes functionality and information density, ensuring that critical data is always visible and accessible.

## 2. Interaction Model
*   **Keyboard-Centricity:** All core functionality should be accessible via keyboard shortcuts. Visual feedback should clearly indicate the current focus and any active shortcuts.
*   **Efficiency over Decoration:** Design choices should focus on speeding up the user's workflow rather than visual flair.

## 3. Messaging & Tone
*   **Direct & Utility-Focused:** All UI text, tooltips, and notifications should be concise and focused on the task. Avoid conversational or overly decorative language.
*   **Industry Precision:** Use accurate freight brokerage terminology to ensure clarity for professional users.

## 4. Design Principles
*   **Contextual Awareness:** The application should anticipate user needs based on their current focus or activity.
*   **Responsiveness:** The side panel interface must remain responsive and performant even with frequent recalculations.
